This is the React single-page frontend for UNI EATS, a campus-focused restaurant discovery app.

## Quick start

```bash
cd frontend
npm install
npm start
```

The app will start in development mode on http://localhost:3000.

Data is currently loaded from static files in `src/data`. 
Backend and database folders at the project root are placeholders for future Flask + PostgreSQL integration.
